List View

Hover over image to enlarge

This view type features parameters and values from SVObserver in list format. You can view the parameters available for the whole tool set and you can rename them by specifying aliases.

The list view includes the following specific components:

  1. Parameter list
  2. List controls (bottom left)

If you click on one of the headers "Name" or "Value", the list elements are ordered alphabetically. If you click again, the order is reversed.

Parameter list

The parameter list shows the parameters you selected.

Each list item indicates the fully qualified name for one parameter. You can specify for each parameter an alias name and rename the parameter.

You can select up to 100 parameters by checking the checkboxes next to them. If you click on the toggle button, you see the values for the selected parameters. For more information, see Show values of selected parameters. You can also change the order of the list entries.

List View Control

The list view offers the following controls:

Element Description

Toggle between visualization of all available parameters and visualization of values for selected parameters
Toggle between alias and fully qualified SVObserver name
Show values of selected parameters

→ Scroll in the selection tree to the inspection tool for which you want to display the images.

→ Click on the plus button to expand the lines below.

→ Click in the check box next to the parameter you want to display.

→ Click on the button Save .
Hide values of selected parameters

→ Click on the button Save .

→ The parameter list containing all parameters available for the inspection is shown.
Show/hide aliases

→ If you want to show the aliases you specified, click on the button Show aliases. Parameters without any alias still show their fully qualified name.

→ If you want to hide the aliases and show the fully qualified name of the parameters, click on the button Hide aliases .

Reorder the entries in the list

→ Click on the small dotted grid and hold the button pressed.

→ Drag the list entry to the desired place and release the button.

Dialog "Set Alias"

Hover over image to enlarge

In the dialog Set alias for dotted name you can rename a parameter. You can specify an alias for each available language that is displayed when you select the respective language in the top bar. You can enter any letter or number and spaces. Aliases are only available in list views, they are not available in other view types nor in SVObserver.

Rename parameters

→ If you selected parameters and see the list of parameters and values, click on the button Set alias for dotted name.

→ In the new window, you can specify an alias for each available language.

→ Click on the button Save .

→ The alias is displayed immediately. You may need to press the toggle button Show aliases to see the aliases.

If you set an alias only for one language, the parameter shows the fully qualified name when the other language is selected.

Dotted name shows the fully qualified name from SVObserver. Value shows the current value resulting from the inspection.

If you set an alias in a list view for one inspection, this alias is not available in views of another type for the same inspection. You must search for the fully qualified name in other view types.

But this alias is shown in other list views for the same inspection. This means it is not possible to create several list views and to give different aliases for the same parameter.
