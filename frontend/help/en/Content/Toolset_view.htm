Tool Set View

The tool set view displays a list of the tools used in the corresponding SVObserver inspection. Currently, it offers a restricted configuration of the tools and their settings as sub-nodes, which represent the titles of tabs known from SVObserver.

For now, the tool set view supports only the configuration of the Math Tool. All other tools must be configured in SVObserver.

You can add and delete tools, copy or cut a tool and paste it in the tool set. You can collapse and expand the sub-nodes for all tools.

You can click on each tool to expand a list of its settings. You can filter the tools if you use the fuzzy search in the top bar.

All tools are available, except the Draw Tool.

If the inspection uses grouped tools, the tool groups are not displayed.

Add tool set to tool set view

→ In an empty view, click on Toolset, and select the desired inspection.

→ The tool set is displayed as a list.

How to add a tool to the tool set

→ Click on an already existing tool to highlight it.

→ Click on a symbol in the left column.

→ The tool is added after the highlighted tool.

How to copy a tool and paste it in the tool set

→ Click on an already existing tool.

→ Click on the copy symbol at the bottom of the view.

→ Click on a tool to highlight it.

→ Click on the paste symbol at the bottom of the view.

→ The tool is added before the highlighted tool.

 

Tool set view and SVObserver

If you add or delete a tool or change the configuration of a tool in SVOWebApp, the changes are automatically and instantly synchronized with SVObserver. Changes in SVObserver are synchronized with SVOWebApp.

The tool set view displays the tools that are added in SVObserver to the configuration. To display the details of each tool, you need a Parameter View for the same inspection that is connected with the tool set view. When you navigate through the tool set, you can click on a node to expand its sub-nodes. When you click on one sub-node, the details are displayed in the Parameter View.

Tool Set View Control

The list view offers the following controls:

Element Description

Copy a tool and place it to the clipboard.
Remove a tool from the tool set and place it to the clipboard.
Paste the tool placed to the clipboard before the highlighted tool.
Expand the nodes from all tools to display the sub-nodes.
Collapse the nodes from all tools to hide the sub-nodes.
Delete the highlighted tool.
