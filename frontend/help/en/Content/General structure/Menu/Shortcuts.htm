Shortcuts

You can use the following keyboard shortcuts to perform different tasks in the SVOWebApp.

Task to Shortcut

Task Shortcut Description
Contextual Help F1 Access a suitable help topic related to the screen subject
Full screen Ctrl + F

Click in the content of a view and press Ctrl + F. The active view is displayed in full screen respectively a view which is already in full screen is set back to normal view.

 
Reload F5 SVOWebApp will be restarted and its content updated
