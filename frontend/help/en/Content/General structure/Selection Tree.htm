Selection Tree

In the selection tree, you can select the tool results from the inspection tool set or an inspection tool. These results refer to the inspection of the SVObserver configuration. The selection tree has a different look depending on the view type.

The selection tree consists of a filter and the tree. You can use the filter or navigate through the tree to find tool results.

Navigate through the tree

There are two ways how you can navigate through the tree. You can choose to select more or less information:

  • Click on the plus button before a node to expand the subordinate values or sub-nodes.
  • Click on the minus button before a node to collapse the subordinate values or sub-nodes.

Select tool result for use in SVOWebApp

The selection tree has a different look depending on the view type. You can select tool results or an image source.

Select an image for use in an Image View or an Image Strip View

The selection tree is displayed when you add an image source to an image view or an image strip view.

→ Click on the plus button of the desired inspection tool.

→ A subordinate node shows.

→ Click on the plus button before the subordinate node.

→ Click in the field next to the image you want to select.

→ The selection tree disappears. The image view shows <UNAVAILABLE>.

→ Switch to run mode to display the selected image.

Select a tool result for use in Parameter View

The selection tree is displayed when you edit a variable value of a parameter in a parameter view.

→ Click on the plus button of the desired inspection tool.

→ If a subordinate node shows, click on the plus button before the subordinate node.

→ Click in the field next to the variable you want to select.

→ The parameter view appears. The field contains the variable you selected.

Select one or several tool results for display in List View

The selection tree is displayed when you add a list view. You can select one, several or a group of tool results.

→ Click on the plus button of the desired inspection tool.

→ If a subordinate node shows, click on the plus button before the subordinate node.

→ Click in the field next to one or several tool results you want to select.

→ A checkmark shows in the field next to the selected tool result .

→ If you want to display all tool results from a node, click on the field next the node you want to select.

→ A checkmark shows in the field next to the selected node .
