View Types

The view type determines the means of how the desired view content, such as data, parameters and images, is visualized.

You can choose from the following available view types:

  • Image View

  • Parameter View

  • Image Strip View

  • List View

  • Toolset View

When you add content to a view, you must select the view type.

You can find more information on how to select a view type to an empty view under: View
