Toggle button to pause or resume inspections

With this toggle button you can pause or resume all or individual inspections in the view set. The button Inspection state shows how many inspections of the configuration are running. By clicking on the button, a drop-down list appears.

You can pause or resume all or individual inspections in this drop-down list. The drop-down list shows only the inspections that are part of the SVObserver configuration. A paused inspection continues to run in SVObserver.

When you are in Edit or in Stop mode, no inspection is active. When you press on the button to resume inspections when in Edit or in Stop mode, this has no effect.

 

Pause all inspections

→ Press the button Inspection state to open a drop-down list.

→ Press the button Pause all inspections.

→ All views pause at the currently displayed image and the corresponding inspection results.

→ The inspections continue to run in the background when you are in Run mode. You can verify in SVObserver that the inspections were not paused.

→ The notification "Reject stream was paused" shows in the top bar.

→ Images resulting from the ongoing inspections are not buffered and are not shown in the Image Strip View when you press Play.

Pause an individual inspection

→ Press the button Inspection state to open a drop-down list.

→ Select the inspection you want to pause.

→ Press the button Pause in front of the inspection's name.

→ The views for this inspection pause at the currently displayed image and the corresponding inspection results.

→ The inspection continues to run in the background when you are in Run mode. You can verify in SVObserver that the inspection was not paused.

→ The notification "Reject stream was paused" shows in the top bar.

→ Images resulting from the ongoing inspection are not buffered and are not shown in the Image Strip View when you press Play.

Resume all inspections

→ Press the button Inspection state to open a drop-down list.

→ Press the button Resume all inspections.

→ All inspections resume with the current image and inspection results. Please note that the inspections continued to run in SVObserver after you had pressed Pause all inspections in the top bar as long as the system was in Run mode.

→ Images that were taken during a pause are not shown in the image strip view. Thus, results to these images cannot be displayed in the List View.

Resume an individual inspection

→ Press the button Inspection state to open a drop-down list.

→ Select the inspection you want to resume.

→ Press the button Resume in front of the inspection's name.

→ The views for this inspection resume with the current image and inspection results. Please note that the inspection continued to run in SVObserver after you had pressed Pause all inspections in the top bar as long as the system was in Run mode.

→ Images that were taken during a pause are not shown in the image strip view. Thus, results to these images cannot be displayed in the List View.

Interaction between image strip and inspection state

It is also possible to pause and resume an individual inspection by clicking on an image shown in image strip view. Alternatively, you can click on the Play/Pause button at the bottom of the image strip view. In both cases, you can see in the top bar the number representing the running inspections change accordingly.
