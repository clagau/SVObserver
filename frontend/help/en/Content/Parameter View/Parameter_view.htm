Parameter View

This view type indicates parameters and their values in reference to a connected view. The parameter view is context sensitive, as this view always displays the parameters that are applied to another view relating to the same inspection.

For example, a parameter view that is targeted to an instance of image view includes all relevant parameters to manage one or more custom overlays.

Or a parameter view relating to a tool set view includes all parameters of a selected Math Tool of the connected tool set view.

 

If you add only a parameter view, the parameter view is empty and displays "Please select a source for this parameter view." You must associate the parameter view with another view type for the same inspection.

 

Associate parameter view and another view

You can establish the dependency between one instance of parameter view and several instances of another view referring to the same inspection by following these steps:

→ Add a parameter view for an inspection.

→ Add the instances of another view type for the same inspection.

→ Click on the instance of the other view. A green frame is displayed to show that this view is selected.

→ You can now edit the parameters in the parameter view.

You only need one parameter view for one inspection. You can switch between several views. The content of the parameter view changes accordingly with the view type.

Currently, parameters for the following views or tools are displayed:

  • Custom Overlay Parameters of an Image View
  • Math Tool (by using a Tool Set View)
