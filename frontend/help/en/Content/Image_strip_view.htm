Image Strip View

This view type features images from an inspection or from a specific inspection tool in sequence. An image is added at the most left place, previous images move to the right when a new image is added to the strip. Image strip view only adds an image of the current object to the image strip when the reject condition specified in the SVObserver's monitor list is fulfilled (i.e. when the result is TRUE). If there is no reject condition specified, no images are displayed.

The most recent image is featured as the first image on the left.

You can select whether you want to display the image containing the overlays of the tool set or the image resulting from one tool.

The image strip view includes the following specific components:

  1. Image strip
  2. Image strip controls

You can navigate through the image strip by dragging the scroll bar or by pressing the arrow keys on your keyboard.

Select Image

You can select whether you want to display images from an inspection or from a specific inspection tool.

Images from an inspection represent the image that results from the SVObserver configuration of this inspection tool. Images from a specific tool represent the applied SVObserver configuration of this tool (filter, threshold etc. included).

This may lead to the effect that images from a specific tool may look different than the image from the whole inspection or from another tool of the same inspection.

 

You can find more details on how to configure inspection tools and overlays in the SVObserver manual. 

 

Add image source to image strip

→ In an empty view, click on the button Image Strip and select the desired inspection.

→ Scroll in the selection tree to the inspection tool for which you want to display the images.

→ Click on the plus button to expand the line below. Click on the plus button in the line below.

→ Click in the check box next to the image you want to display.

→ Switch the operating mode to Run Mode.

In order to find inspections faster, you can use the filter. Click in the filter field at the top, and enter your search term.

You can find more details on how to use the filter and the displayed results in Using the Filter. 

Image Strip Controls

The image strip view offers the following controls:

Element Description
Play/Resume inspection
Pause inspection

Interaction between image strip view and image view

If you click on an image in the image strip view, the linked image view automatically shows the corresponding image. If you pause or resume an inspection, images in image strip view and image view are paused or resumed as well. For an instruction on how to pause or resume an inspection, see below.

 

Pause inspection

→ In the view menu, press the button Pause or click on one of the currently displayed images

→ The image strips pauses at the currently displayed image. The image in an image view (if applicable) and its corresponding inspection results are paused as well.

→ The number of running inspections displayed in the top bar next to Inspection state changes accordingly.

→ The inspection continues to run. You can verify in SVObserver that the inspection was not paused.

→ The notification "Reject stream was paused" shows in the top bar.

→ Images resulting from the ongoing inspection are not buffered and are not shown when you resume the inspection by pressing Play or one of the displayed images in the image strip view.
Resume inspection

→ In the view menu, press the button Play .

→ The image strip resumes at the image that was taken last in the inspection. Please note that the inspection continued to run in SVObserver after you had pressed Pause in the image strip.

→ The number of running inspections displayed in the top bar next to Inspection state changes accordingly.

→ The image in an image view (if applicable) and its corresponding inspection results resume to update as well.

→ The notification "Reject stream was resumed" shows in the top bar.

→ Images that were taken during a pause are not shown in the image strip view.
Show image in bigger size

→ Right-click on the image and select Open image in new tab.

→ A new tab opens where you can see the same image in bigger size.

Save image

→ Right-click on the image and select Save image as....

→ Define a file name, file extension and a location.

→ Press Save.
