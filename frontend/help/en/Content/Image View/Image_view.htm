Image View

Hover over image to enlarge

This view type features images acquired by SVObserver. It contains overlays from the selected inspection or tool.

The image view includes the following specific components:

  1. Featured image (center), depicted with overlays
  2. Image controls (bottom left)
Note

You can save all view sets and incorporated views via the view manager. All changes will only be saved in the SVOWebApp configuration.

Select Image 

You can select whether you want to display images from an inspection or from a specific inspection tool.

The image from an inspection includes all inspection overlays.

Images from an inspection represent the image that results from the SVObserver configuration of this inspection tool. An image from a specific tool represents the applied SVObserver configuration of this tool (filter, threshold etc. included).

This may lead to the effect that an image from a specific tool might look different than the image from the whole inspection or from another tool of the same inspection.

 

You can find more details on how to configure inspection tools and overlays in the SVObserver manual. 

 

Add image source to image view

→ In an empty view, click on Image , and select the desired inspection.

→ Scroll in the selection tree to the inspection tool for which you want to display the images.

→ Navigate through the selection tree by clicking on the plus buttons when needed.

→ Click in the check box next to the image you want to display.

→ Switch the operating mode to Run Mode.

In order to find inspections faster, you can use the filter. Click in the filter field at the top, and enter your search term.

You can find more details on how to use the filter and about the displayed results in Using the Filter. 

Overlays

An overlay is a shape superimposed on an image displayed in image view. There are two different types of overlays. Inspection overlays and custom overlays.

Inspection overlays are transferred from the SVObserver inspection tools.

You can create custom overlays to highlight areas in the Parameter View. You can hide or show custom overlays.

 

You can find more details about overlays and the difference between custom overlays and inspection overlays in Overlays.

 

Dragging and Panning Images

You can navigate the image within the view by using the mouse or your fingers.

Zoom image in image view

→ Place the cursor on the image.

→ Use the mouse wheel to zoom the image.

Alternatively, you can spread your fingers apart if your device support multi-touch.
Drag image in image view

→ Place the cursor on the image.

→ Hold down the mouse button and drag the image.

Alternatively, you can pinch your fingers together if your device support multi-touch.

Image Controls

The image view offers the following elements to control the image (from left to right):

Element Description
1:1 original image size
Fit image to width
Fit image to height
Download image
Download image with overlays

Display overlay list to show or hide custom overlays all together or individually

Display image information

Show image width/height, cursor position (X, Y) relative to image, color and RGB color value of the pixel
Download image without overlays

You can download the image without any overlays to any location

→ In the view menu, press Download image .

→ The image is saved to the location specified in the browser settings. The saved image does not contain any inspection or custom overlays.
Download image with overlays

You can download the image with inspection and custom overlays to any location

→ In the view menu, press Download image with overlays .

→ The image is saved to the location specified in the browser settings.

Overlay List

The overlay list offers options to show and hide custom overlays in the image.

You can show or hide custom overlays either all together or individually.

→ To open the overlay list, click on the button Overlay list in the view controls bar.

The overlay list opens in a separate window within the active view next to the featured image.

Hide custom overlays in image view

You have two options to hide custom overlays in the featured image:

Hide all custom overlays

→ In the view menu, click on the button Overlay list .

→ The overlay list opens within the active view.

→ Uncheck the checkbox next to the heading Overlays.

→ Custom overlays are no longer displayed with the featured image.

Hide individual custom overlays

→ In the view menu, click on the button Overlay list .

→ The overlay list opens in the active view.

→ Uncheck the checkbox next to the custom overlay you want to hide .

→ The respective custom overlay is no longer displayed with the image.
Show custom overlays in image view

You have two options to display custom overlays in the featured image:

Show all custom overlays

→ In the view menu, click on the button Overlay list .

→ The overlay list opens in the active view.

→ Activate the checkbox next to the heading Overlays.

→ Custom overlays are displayed in the image.

Show individual custom overlays

→ In the view menu, click on the button Overlay list .

→ The overlay list opens in the active view.

→ Check the checkbox next to the custom overlay you want to show .

→ The respective custom overlay is displayed in the image.

Image Information

You can display additional information about the image and the cursor position.

→ To display image information, click on the button Image information in the image controls bar.

The information shows in a separate window within the active image view.

You can view the dimensions of the image (width/height), cursor position (X/Y) relative to image, and color and RGB color value (0, 0, 0) of the pixel.

Image View during Run Mode

The image shown in an Image View is constantly updated when in Run Mode. The image view shows in Run Mode always the current image of the SVObserver configuration.

 

Select image to display in Image View from Image Strip View

If you have an image view and an image strip view, both referring to the same inspection, you can select an image from the image strip to be displayed in the image view.

→ You need an Image Strip View and an Image View for the same inspection.

→ Click on an image in the image strip.

→ An image of the corresponding object is displayed in the image view.

→ The image strip is paused and the notification "Reject stream was paused" shows in the top bar.
