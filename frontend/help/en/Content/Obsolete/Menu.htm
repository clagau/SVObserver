Menu

You find the menu on the right side of the bottom bar.

Hover over image to enlarge

→ Click on the button Menu to access the menu items.

→ Click on the desired menu item.

The following menu items are available:

Create, rename, save and delete view sets

You can find more information about this menu point under View Manager

Access a full list of keyboard shortcuts

You can find more information about this menu point under Shortcuts

Log off from SVOWebApp

Access SVOWebApp help
Shows information about the versions of SVOWebApp, SVObserver and SVOGateway.
