Image View

Click image to enlarge     

 

This view type features images acquired by the SVObserver and contains overlays by default.

The image view includes the following specific components:

  1. Featured image (center), depicted with or without overlays
  2. Image controls (bottom left)

Note: You can save all view sets and incorporated views via the view manager. All changes will only be saved in the SVOWebApp configuration.

 

Featured Image

If you add an image view to the view set, you have to choose which image should be displayed in the corresponding view.

→ To select a source, click on the desired tool image listed in the center of the view

Click image to enlarge     

 

 

Overlays

An overlay is a shape superimposed on an image displayed in image view. You can hide or show overlays.

 

Dragging and Panning Images

You can navigate the image within the view using the mouse.

Multi-touch is not supported yet.

Zoom image in image view

→ Place the cursor on the image

→ Use the mouse wheel to zoom the image
Drag image in image view

→ Place the cursor on the image

→ Hold down the mouse button and drag the image

You can save the image without applied overlays to any location

  • In the view menu, press Save without overlays

    —The image will be saved to the location as specified in the browser settings
Save image with overlays to local repository

You can save the image with its applied overlays to any location

  • In the view menu, press Save with overlays

    —The image will be saved to the location as specified in the browser settings

 

Images with Custom Overlays

A custom overlay is a customizable overlay that you can apply and superimpose on an image in image view. It is possible to apply multiple custom overlays to one instance of the image view.

You can use an overlay for various purposes, e.g. to identify or indicate errors. You can feature certain aspects within the defined section of the image.

The custom overlay is only available with the SVOWebApp.

The overlays' parameters are managed in a separate instance of a parameter view.

 

You can find more details on how to use the parameter view under: Parameter View 

 

Overlay List

The overlay list offers options to show and hide overlays with the image.

You can show or hide overlays either all together or individually.

→ To open the overlay list, click on the button Overlay list in the view controls bar.

The overlay list opens in a separate window within the active view next to the featured image.

 

Overlay List Elements

The overlay list offers the following controls (from left to right):

Element Description

[Label]

 Overlay name
Hide individual overlay
Show individual overlay
Hide all overlays on featured image
Show all overlays on featured image
Hide overlays in image view

You have two options to hide overlays in the featured image:

Hide all overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will open within the active view

  2. Click on the button Hide all overlays

    — The overlays will no longer be displayed with the featured image

Hide individual overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Uncheck the checkbox next to the overlay you want to hide

    —The respective overlay will no longer be displayed with the image
Show overlays in image view

You have two options to display overlays in the featured image:

Show all overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will open in the active view

  2. Click on the button Show all overlays

    — The overlays will be displayed with the image

Show individual overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Check the checkbox next to the overlay you want to show

    —The respective overlay will be displayed with the image

 

Image Information

You can display additional information about the image and the cursor position.

→ To display image information, click on the button Image information in the image controls bar.

The information shows in a separate window within the active image view.

You can view the size of the image (width/height), cursor position (X/Y) relative to image, and color and RGB color value (0, 0, 0) of the pixel.
