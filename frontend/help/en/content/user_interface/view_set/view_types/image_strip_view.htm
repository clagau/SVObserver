Image Strip View

This view type features images (e.g. reject images) in sequence.

The image strip view includes the following specific components:

  1. Image strip
  2. Image strip view controls

 

Image Strip

→ To select a source for the image strip, click on the desired tool image listed in the center of the view

→ Select a value for the remote input

The most recent image will be featured as the first image on the right.

 

Image Strip View Controls

The image strip view offers a set of view type specific controls:

Element Description
Play
Pause
Select input value
Pause image strip

  • In the view context menu, press button Pause or click on one of the currently displayed images

    — The images will stop moving. Images taken after pressing Pause will be displayed when you press Play again
Play image strip

  • In the view context menu, press the button Play

    — The images will move in the direction from right to left. The image strip resumes with the image that was taken first after pressing Pause

Select input value
  • In the view context menu, press the button Settings

    — The list of possible input values will be displayed
