List View

Click image to enlarge     

 

This view type features parameters in list format. You can view the parameters available for a selected group, e.g. for a selected inspection.

The list view includes the following specific components:

  1. Parameter list
  2. List controls (bottom left)

 

Parameter list

The parameter list shows all available parameters for the selected group.

Each list item indicates the fully qualified name for one parameter.

You can select parameters by checking the checkboxes next to them. You see the values for the selected parameters. For more information, see Show values of selected parameters

 

List View Control
