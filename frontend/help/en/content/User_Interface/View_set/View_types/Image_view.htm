Image View

This view type features images acquired by the SVObserver and will be indicated with overlays by default.

The image view includes the following specific components:

  1. Featured image (center), depicted with or without overlays
  2. Image view controls (bottom left)

Note: You can save all view sets and incorporated views via the view manager. All changes will only be saved in the SVOWebApp configuration.

 

Featured Image

If a view of the type image view has been added to the view set, the user will have to choose a display image. The selected image will then be featured in the active view.

→ To select a source, click on the desired tool image listed in the center of the view

 

Overlays

An overlay is a shape superimposed on an image displayed in image view. The user can hide or show overlays.

 

Dragging and Panning Images

The user can navigate the image within the view using the mouse.

Multi-touch is not supported yet.

Zoom image in image view

  1. Place the cursor on the image

  2. Use the mouse wheel to zoom the image
Drag image in image view

  1. Place the cursor on the image

  2. Hold down the mouse button and drag the image

  • In the view context menu, press Save without overlays

You can save the image with its applied overlays to any location

  • In the view context menu, press Save with overlays

    —The image will be saved to the location as specified in the browser settings

 

Images with Custom Overlays

A custom overlay is a customizable overlay that can be applied and superimposed on an image featured in image view. It is possible to apply multiple custom overlays to one instance of image view.

The overlay can be used for various purposes, e.g. to identify or indicate errors. It can also be used to feature certain aspects within the defined section of the image.

The custom overlay is only available with the SVOWebApp.

The overlays' parameters are managed in a separate instance of a parameter view.

 

For more details on how to use the parameter view, read topic: Parameter view 

 

Overlay List

The overlay list offers options to show and hide overlays with the image.

Overlays can be shown or hidden either all together or individually.

→ To open the overlay list, click on button Overlay list in the view controls bar.

The overlay list will open in a separate window within the active view next to the featured image.

 

Overlay List Elements

The overlay list offers the following controls (from left to right):

Element Description

[Label]

 Overlay name
Hide individual overlay
Show individual overlay
Hide all overlays on featured image
Show all overlays on featured image
Hide overlays in image view

There are two options for hiding overlays in the featured image:

Hide all overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will open within the active view

  2. Click on button Hide all overlays

    — The overlays will no longer be displayed with the featured image

Hide individual overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Uncheck the checkbox next to the overlay to be hidden

    —The respective overlay will no longer be displayed with the image
Show overlays in image view

There are two options to display overlays in the featured image:

Show all overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Click on the button Show all overlays

    — The overlays will be displayed with the image

Show individual overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Check the checkbox next to the overlay to be hidden

    —The respective overlay will be displayed with the image

 

Image Information

Information about the image and the cursor position can be viewed.

→ To display image information, click on the button Image information in the image view controls bar.

The information will be displayed in a separate window within the active image view.

The user can view the size of the image (width/height), cursor position (X/Y) relative to image, and color and RGB color value (0, 0, 0) of the pixel.
