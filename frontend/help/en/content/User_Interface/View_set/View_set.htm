View Set

A view set holds one or more views. These views are grouped together by the user to create a custom joint display of data, parameters and images. The user can arrange and resize each view as needed. The organization of the featured views and the layout are saved. The above image shows an active view set with four included views.

If there is no view set created by the user, an empty view will be displayed at the center of the screen.

The SVOWebApp configuration can include multiple view sets. The user can choose which of the available view sets is to be displayed in the center of the page by using the view set navigation in the bottom bar.
