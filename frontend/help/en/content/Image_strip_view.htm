Image Strip Controls

The image strip view offers the following controls:

Element Description
Play
Pause
Select input value
Pause image strip

  • In the view menu, press the button Pause or click on one of the currently displayed images

    — The images will stop moving. Images taken after pressing Pause will be displayed when you press Play again
Play image strip

  • In the view menu, press the button Play

    — The images will move in the direction from right to left. The image strip will resume with the image that was taken first after pressing Pause

Select input value
  • In the view menu, press the button Settings

    — The list of possible input values will be displayed
