Image View

Hover over image to enlarge     

 

This view type features images acquired by the SVObserver. It contains overlays from the selected inspection or tool.

The image view includes the following specific components:

  1. Featured image (center), depicted with overlays
  2. Image controls (bottom left)

Note: You can save all view sets and incorporated views via the view manager. All changes will only be saved in the SVOWebApp configuration.

 

Select Image 

You can select whether you want to display images from an inspection or from a specific inspection tool.

The image from an inspection includes all inspection overlays.

Images from a specific inspection tool represent the image that results from the SVObserver configuration of this inspection tool. Images can look different from images due to applied SVObserver configuration (e.g. filters, thresholds).

 

You can find more details on how to configure inspection tools and overlays in the SVObserver manual. 

 

Add image source to image view

→ In an empty view, click on the button "Add view" in the upper right corner, and select the desired inspection and "Image".

→ Scroll in the selection tree to the inspection tool for which you want to display the images.

→ Navigate through the selection tree by clicking on the plus buttons when needed.

→ Click in the check box next to the image you want to display.

→ Switch the operating mode to Run Mode.

 

In order to find inspections faster, you can use the filter. Click in the filter field at the top, and enter your search term.

You can find more details on how to use the filter and about the displayed results in Using the Filter. 

 

Overlays

An overlay is a shape superimposed on an image displayed in image view. You can hide or show overlays. There are two different types of overlays. Inspection overlays and custom overlays.

Inspection overlays are transferred from the SVObserver inspection tools.

You can create custom overlays to mark areas in the Parameter View.

 

You can find more details about overlays and the difference between custom overlays and inspection overlays in Overlays.

 

Dragging and Panning Images

You can navigate the image within the view using the mouse.

Multi-touch is not supported yet.

Zoom image in image view

→ Place the cursor on the image

→ Use the mouse wheel to zoom the image
Drag image in image view

→ Place the cursor on the image

→ Hold down the mouse button and drag the image

 

Image Controls

Save featured image with overlays to local file system

Display image information

Save image without overlays to local repository

You can save the image without any overlays to any location

  • In the view menu, press Save

    —The image will be saved to the location as specified in the browser settings. The saved image does not contain any inspection or custom overlays.
Save image with overlays to local repository

You can save the image with inspection and custom overlays to any location

  • In the view menu, press Save with overlays

    —The image will be saved to the location as specified in the browser settings.

 

Overlay List

The overlay list offers options to show and hide custom overlays in the image.

You can show or hide custom overlays either all together or individually.

→ To open the overlay list, click on the button Overlay list in the view controls bar.

The overlay list opens in a separate window within the active view next to the featured image.

 

If you click on the header "Overlays", the overlay list is ordered alphabetically. If you click again, the order is reversed.

 

Overlay List Elements

The overlay list offers the following controls (from left to right):

Element Description

Label

 Name of custom overlay
Hide individual custom overlay
Show individual custom overlay
Hide all custom overlays on featured image
Show all custom overlays on featured image
Hide custom overlays in image view

You have two options to hide custom overlays in the featured image:

Hide all custom overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will open within the active view

  2. Click on the button Hide overlays

    — Custom overlays will no longer be displayed with the featured image

Hide individual custom overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Uncheck the checkbox next to the custom overlay you want to hide

    —The respective custom overlay will no longer be displayed with the image
Show custom overlays in image view

You have two options to display custom overlays in the featured image:

Show all custom overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will open in the active view

  2. Click on the button Show overlays

    — Custom overlays will be displayed with the image

Show individual custom overlays

  1. In the view menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Check the checkbox next to the custom overlay you want to show

    —The respective custom overlay will be displayed in the image

 

Image Information

You can display additional information about the image and the cursor position.

→ To display image information, click on the button Image information in the image controls bar.

The information shows in a separate window within the active image view.

You can view the size of the image (width/height), cursor position (X/Y) relative to image, and color and RGB color value (0, 0, 0) of the pixel.

 

Image View during Run Mode

The image shown in an Image View is constantly updated when in Run Mode. The image view shows in Run Mode always the current image of the SVObserver configuration.

You can pause the image shown in the Image View:

→ You need an Image Strip View.

→ Press Pause in the Image Strip View.

→ The notification "Reject stream was paused" shows in the top bar.

→ The inspection continues to run. You can verify in SVObserver that the inspection was not paused.

 

For more information about how to resume the automatic update of images, refer to Play image strip.
