General Information

The SVOWebApp is provided by a web server on the SVIM. It retrieves data and images from SVObserver to present them in an internet browser.

The SVOWebApp uses the HTML5 standard to support a multilingual, modern and user-friendly interface. It is embedded in the machine HMIHuman machine interface. The web app can also be accessed by the user on multiple devices, e.g. a laptop or other mobile devices, via the internet browser, regardless of the operating system used.

The implementation of the current SVOWebApp is still in progress. All settings are still supported in the current SVObserver GUIGraphical user interface. The final goal is to provide the user with the required functionality known from the current SVObserver GUI.
