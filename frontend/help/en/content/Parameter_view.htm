Parameter View

This context sensitive view type indicates parameters and their values in reference to a selected target featured in another view. The parameter view is context sensitive, as this view always displays the parameters that are relevant to a target content.

For example, a parameter view that is targeted to an instance of image view includes all relevant parameters to manage one or more custom overlays.

 

Custom Overlay Parameters

You can parameterize custom overlays in a parameter view. The target image and the overlay are featured in an image view. You can apply multiple custom overlays to one instance of an image view.

You can find more details on custom overlays in general and on how to apply custom overlays in an image view under: Image View  

The parameter view for overlays includes the following specific components:

  1. Overlay list
  2. Overlay parameters
  3. Parameter controls

 

Overlay List

This list indicates all available overlays for the target image.

Note: There might be no entry in the overlay list until you have created a custom overlay.

The overlay list includes the following elements:

Element Description

[Label]

 Overlay name
Show or hide individual overlay
Edit overlay

 

Edit parameters for an overlay

→ Click on the button Edit overlay in the overview list

→ Click in the parameter field to enter a value or input parameter from the parameter list

— For more information about each parameter, see Overlay Parameters

 

Overlay Parameters

There are various input fields to add parameters values for the overlay.

Each input field includes the following elements:

Element Description

[PARAMETER NAME]

 Parameter value to be entered
Reset parameter values to the one previously saved

Certain parameter fields allow for entering values. In fields which name contains "variable", input parameters are selected from the parameter list.

Following overlay parameters are available:

Parameter field Description
[Overlay name] Overlay name
X X coordinate value [px] of the shape
X Variable

X variable as selected from the parameter list
OFFSET LEFT Left position of X [px] relative to the left edge of the featured image
OFFSET LEFT VARIABLE Offset left variable as selected from the parameter list
Y Y coordinate value [px] of the shape
Y Variable Y variable as selected from the parameter list
OFFSET TOP Top position of Y [px] relative to the top edge of the featured image
OFFSET TOP VARIABLE Offset top variable as selected from the parameter list
WIDTH Width of the shape [px]
WIDTH VARIABLE Width variable as selected from the parameter list
HEIGHT Height of the shape [px]
HEIGHT VARIABLE Height variable as selected from the parameter list
COLOR Hexadecimal color of the shape, e.g. #ebfc02
COLOR VARIABLE Color variable as selected from the parameter list
LINE STYLE

Line style of the rectangle [px]. The following line styles can be applied:

  • line
  • dotted

 

Apply Values from Parameter List

If there is an overlay tool provided for the custom overlay, the overlay of this tool, including the optional analyzer, can be superimposed on the target image.

Each list item indicates the full path of the input parameter.

→ Click in the parameter field to open the parameter list

→ Activate the radio button next to the desired input parameter

→ Click on the button Close in the top right corner

The input parameter are instantly applied to the selected overlay. The actual parameter value is not indicated in the parameter list. In some cases, it might be useful to view the value referring to the applied parameter. You have to create a view of the type list for the same inspection. You can find more information about how to display the parameter values under Show values of selected parameters in List View.

You can select the parameters applied to the overlay and display them together with their values.

 

You can find more information on list view under: List View

You can find more information on how to create a view under: View 

 

Parameter Controls

The parameter view offers the following elements to control the parameters (from left to right):

Element Description

Add overlay
Hide all overlays
Show all overlays

Delete overlay

Note: Clicking this button deletes the overlay that is currently selected. The overlay is no longer available for future use and the parameter set is permanently erased.
Add custom overlay

You can create as many overlays for a target image as desired by adding them to the overlay list in the respective parameter view.

  • In the view menu, click on the button Add overlay

    — A new overlay will be added to the overlay list

Delete custom overlay

Besides the option of hiding an overlay in the image view, you can also delete an overlay.

  1. In the overlay list, click on the button Edit overlay next to the overlay (label) you want to delete

    — The parameters are displayed right next to the overlay list

  2. In the view menu, click on the button Delete overlay

    — The overlay and its parameters are deleted

 

Target Image

The dependency between the overlay parameters and the target image featured in an image view is visualized by a green frame around the target image view.

To view one or more overlays on an image as well as the overlay parameters, following views are required:

  1. One instance of image view that features the target image as well as the custom overlays

  2. One instance of parameter view to edit the overlay parameters

Both instances of image view and parameter view have to refer to the same inspection as indicated in the view title

Associate parameter view and image view

You can establish the dependency between one instance of parameter view and image view by following these steps.

  1. Click on the instance of the parameter view

  2. Create and/or edit custom overlays as described in section Parameter Controls

  3. Click on the instance of the image view featuring the target image

    — The custom overlays will be superimposed on the image in view
