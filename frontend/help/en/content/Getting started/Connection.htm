Connection

The SVOWebApp can connect to the inspection system via IP address or the computer name and port 80.

The connection to the inspection system must run via the protocol SVOGateway. It allows for the communication between SVObserver and clients, such as machine HMI and internet browser.

Note: SVObserver is a high-performance inspection system. Connections to other networks then to the internal machine network are permitted. It is not allowed to install additional applications or services onto the inspection system. Virus scanners or firewalls are not allowed to run on the inspection system. SVRCSeidenader Vision Remote Control clients need to be compatible to the running version of SVObserver.
