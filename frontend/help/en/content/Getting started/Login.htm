Login

Once connected to the SVOWebApp, the login screen appears in the internet browser.

Enter the username and password used in HMI to log in to the SVOWebApp. The username is used to set the credentials used in the SVOWebApp.

→ Click in the field Username

→ Enter username

→ Click in the field Password

→ Enter password

→ Click the button LOGIN

You can log off from the SVOWebApp via the Menu.
