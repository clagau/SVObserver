Login

Once connected to the SVOWebApp, the login page will appear in the internet browser.

To log in to the SVOWebApp, the username and password used in HMI must be entered. The username is used to set the credentials used in the SVOWebApp.

→ Click in the field Username

→ Enter username

→ Click in the field Password

→ Enter password

The user can log off from the SVOWebApp via the Menu.
