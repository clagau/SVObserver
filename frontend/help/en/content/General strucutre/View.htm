View

Click image to enlarge     

 

A view provides a means of viewing and interacting with data or images via the SVOWebApp. It is the basic element of a view set.

You can find more information about view sets under: View Set

If you have added content to a view, it has the following components:

  1. View title
  2. View controls
  3. View content
  4. View type controls

If you have not yet edited a view, it only contains a single, empty view, which includes only the view controls in the top right corner. The view title as well as the view type controls are shown if you have added content.

Click image to enlarge     

 

 

View Title

The view title indicates the group and the view type of the added view content. If the view is empty, no title is shown.

The view title provides more information about the source and the content type.

 

View Controls

You can add view content via the general view controls located in the upper right corner of the view.

Views include the following general controls:

Element Description
Add view content, such as data, parameters, images
Enlarge view

Split view
Delete view
Add view to view set

You can add as many views to a view set as you desire. The views are initially empty.

  • In the upper right corner of the view, click on Split view

    — The view will be split either on a horizontal or vertical dividing line, depending on the original shape of the view to be split

 

View Content

The actual content of the view may include data, parameters or images which are visualized within the view. The displayed view content depends on the group (e.g. inspection) and the view type (e.g. image, parameter, image strip, etc.) that you selected.

 

You can find more information about view types under: View types  

 

Add view content

You can add view content, such as data, parameters or images to a view by selecting a group (e.g. a specific inspection) and choosing from a list of predefined view types.

Click image to enlarge     

 

You can find more information on view types under: View types 

  1. In the upper right corner of the view, click on Add view

    — A submenu opens. This submenu indicates the available groups, e.g. inspections

  2. Click on the desired item of the group submenu

    Example: The groups submenu indicates various inspections

    — A second submenu opens. This submenu indicates the available view types

  3. Click on the desired item of the view type submenu

    Example: The view type submenu indicates the available view types

    — The view content is displayed in the center of the view

 

View Type Controls

After you added content to the view, you see in the lower bar control elements. These controls vary depending on the view type. The view type controls are explained together with the respective view type.
