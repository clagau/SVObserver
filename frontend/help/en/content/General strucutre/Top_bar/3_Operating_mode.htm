Operating Mode

Click image to enlarge   

 

You can operate the SVObserver in various modes. The choice of mode depends on the required tasks and processes conducted by the SVObserver.

This button in the top bar indicates the mode in which the SVObserver is operating. You can switch the operating mode if you have sufficient rights.

Note: The area in the top bar displays the operating mode that is currently set

 

The following modes are available:

Mode Description
The SVObserver acquires and processes images, reads inputs and sets outputs in this mode
The SVObserver does not accept any triggers while in this mode. You cannot edit or configure tools in this mode
The SVObserver processes previously archived images
You can access some tool adjustments while taking images for testing purposes when you have sufficient rights
You can add, adjust and delete configurations and tools when you have sufficient rights
Switch Operating Mode

To conduct certain user interactions, you have to switch the SVObserver into the required operating mode.

When you change the mode, the SVObserver is executed in the selected mode.

→ Click on the button indicating the respective operating mode currently set

→ Select the designated mode from the available submenu items

→ The operating mode is set accordingly
