View Set Navigation

Click image to enlarge     

 

The view set navigation includes one or more buttons. Each button relates to an existing view set and is labeled with the view set name. The default name for the view set which has not been edited is "Initial".

A gray-shaded button indicates that the referring view set is being displayed at the center of the screen. This view set is referred to as the active view set.

→ Click on the referring view set button to display any other view set in the center of the screen.
