View Manager

You can create, rename, save and delete view sets in the view manager

Click image to enlarge     

 

To open the view manager:

→ Click on the button Menu in the right bottom corner of the screen

→ Click on the menu item View Manager

 

View Manager Controls

The view manager includes the following specific components and controls:

Element Description

Apply changes
Add new view set
Delete view set

Save all view sets

Note: Settings saved via the SVOWebApp interface are only saved in the SVOWebApp configuration and have no effect on SVObserver. If you want to save your settings in the SVObserver configuration, you can do so by using the SVObserver interface.
 
Add new view set

  1. Click on the button Menu (bottom right corner of the screen)

    — The menu opens, indicating a list of menu items

  2. Click on the menu item View Manager

  3. In the input field Name, enter the desired name for the view set you want to create

  4. Click on the button Add View Set

    — A view set button will be added to the bottom bar, indicating the name of the added view set
Rename existing view set

  1. In the view set navigation (bottom bar), click on the button indicating the view set name you want to rename

    — The selected view set will be displayed in the center of the screen. The view set button will be highlighted in gray

  2. Click on the button Menu (bottom right corner of the screen)

    — The menu opens, indicating a list of menu items

  3. Click on the menu item View Manager

    — The view manager opens in a pop-up window

  4. In the input field Name, replace the old title with the desired new title

  5. Click on the button Apply changes

    — The selected view set will be renamed. The respective view set button will indicate the new name

Save all view sets

  • This instruction describes how you can save view sets and included views in the SVOWebApp configuration.

    1. Click on the button Menu (bottom right corner of the screen)

      — The menu opens, indicating a list of menu items

    2. Click on the menu item View Manager

      — The view manager opens in a pop-up window. In the input field "Name", the selected view set will be indicated

    3. Click on the button Save All View Sets

      — All changes to the view sets will be saved in the web app configuration

    Note: Settings saved via the SVOWebApp interface are only saved in the SVOWebApp configuration and have no effect on SVObserver. If you want to save your settings in the SVObserver configuration, you can do so by using the SVObserver interface.

    Delete view set

    1. Click on the view set button in the bottom bar indicating the name of the view set you want to delete

      — The selected view set is displayed in the center of the screen. The view set button will be highlighted in gray

    2. Click on the button Menu (bottom right corner of the screen)

      — The menu opens, indicating a list of menu items

    3. Click on the menu item View Manager

      — The view manager opens in a pop-up window. In the input field "Name", the selected view set will be indicated

    4. Click on the button Delete View Set

      — The selected view set will be removed from your screen
