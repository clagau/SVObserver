View Set

Hover over image to enlarge

A view set is the core part of SVOWebApp and is located in the center of the screen. It contains one or more views. You can group views to create a custom joint display of data, parameters and images. You can arrange and resize each view as needed. The organization of the views and the layout are saved. The above image shows an active view set with four included views.

If you have not yet created a view set, you see an empty view where you can add a view.

Learn how to add content to a view. 

 

The SVOWebApp configuration can include multiple view sets. You can choose which of the available view sets you want to display in the center of the screen by using the view set navigation in the bottom bar.

 

You can find more details on how to create, edit and work with views under: View 

You can find more details on how to create, manage and save view sets under: View Manager 

You can find more details on how to display any of the available view sets under: View Set Navigation
