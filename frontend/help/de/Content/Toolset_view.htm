Toolset View

The toolset view displays a list of the tools used in the corresponding SVObserver inspection. Currently, it offers a restricted view on the tools and their settings as sub-nodes, which represent the titles of tabs known from SVObserver. For now, the Toolset View offers no further interaction with these settings. You can click on each tool to expand a list of its settings.

If the inspection uses grouped tools, the tool groups are not displayed.

Add toolset to toolset view
