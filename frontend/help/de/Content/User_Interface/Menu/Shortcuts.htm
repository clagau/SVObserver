Shortcuts

The following keyboard shortcuts can be used to perform different tasks in the SVObserver web app.

Task to shortcut

Task Shortcut Description
Contextual Help F1 Access web app help for a certain instance of the web app
Full screen Ctrl + F

The active view will be displayed in full screen respectively a view which is already in full screen will be set back to normal view

 
Reload F5 Reload Page
