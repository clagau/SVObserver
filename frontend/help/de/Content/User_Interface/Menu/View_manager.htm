View Manager

The view manager allows the user to create, rename, save and delete view sets.

The view manager can be accessed via the menu in the bottom bar

→ To open the menu, click on the button Menu

→ In the menu, click on the menu item View Manager. The view manager opens in a pop-up window

The view manager includes the following specific components and controls:

Element Description
Name [Input field]

Displays current or default view set name of the active view. The default name is "Inititial".

The user can enter view set name for either new view set or change the name of an existing view set
Add view set
Delete view set

Save all view sets

Note: The view sets and included views will only be saved in the web app configuration. The user who intends to save those changes in the SVObserver configuration will have to save the changes in the configuration in the SVObserver GUI via the file menu (See: SVIM Manual, Chapter 3 - Inspection Configuration)

  
Close view manager
Add View Set

  1. In the input field Name, enter the desired name for the view set to be created

  2. Click on the button Add View Set

    — A view set button will be added to the bottom bar, indicating the name of the added view set

  3. Click on view set button in the bottom bar indicating the name of the added view set

    — The selected view set is displayed in the center of the screen

Delete View Set

  1. In the input field Name, enter the name for the view set to be removed

  2. Click on the button Delete View Set

    — A view set will be removed from your screen

  3. Rename view set

Rename View Set

  1. In the view set navigation (bottom bar), click on the button indicating view set name to be changed

    — This view set will be displayed in the center of the screen, the view set button

  2. In the bottom bar, click on the button Menu

    — The menu opens, indicating a list of menu items

  3. Click on menu item View Manager

    — The view manager opens in a pop-up window

  4. In the input field Name, replace the old title with the new title
