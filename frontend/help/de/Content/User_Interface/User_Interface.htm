User Interface

DEUTSCH

The web app user interface is organized into three main sections:

  1. Top bar

  2. View set

  3. Bottom bar

  4. Menu

Click on image to enlarge

 

Show and Hide the Top or Bottom Bar

The user can choose whether he or she wants to see the top bar in the browser.

 

Hide the Top or Bottom Bar

You can hide the top bar and / or the bottom bar to allow for more screen space.

  1. Mouse over the semi-circular white shape on the bar

    — The button Hidewill slide out

  2. Click on the button Hide

    — The bar will phase out and be no longer displayed in the user interface until you choose to view it when required

Display Hidden Top or Bottom Bar

Currently the only way to un-hide the bar is by refreshing the page in the browser [Shortcut: F5]
