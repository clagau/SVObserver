View

A view provides a means of viewing and interacting with data or images on the Web App. It is the basic element of a view set.

A view set which has not yet been edited will only contain a single, empty view.

The view includes four controls in the upper right corner.

You can add as many views to a view set as you desire. The views will be initially empty.

To learn about how to add content, read instruction: Add content to view

  • In the upper right corner of the view, click on Split view

    — the view will be split either on a horizontal or vertical dividing line, depending on the original shape of the view to be split

Add content to view

You can add content, such as data, parameters or images to a view for a selected group (e.g. a specific inspection) by choosing from a list of predefined view types.

To learn about view types, read section: View type

  1. In the upper right corner of the view, click on Add view content

    — sub menu opens. This sub menu indicates the available groups, e.g. inspections

  2. Click on the desired item of the group sub menu

    — a second sub menu opens. This sub menu indicates the available view types

  3. Click on the desired item of the view type sub menu

    — the view content will be displayed in the view

A view with added content will have the components:

  1. View title

  2. View controls

  3. View object

  4. View context controls
