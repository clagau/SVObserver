Parameter

DEUTSCH

This context sensitive view type displays parameters and values in reference to a selected target featured in another view.

Dependencies with Other View Types

The user can retrieve parameters from the SVObserver in parameter view in reference to a target in another view type.

Example: Overlay parameters

  • One instance of the image view featuring the image with one or more overlays

  • One instance of the parameter view featuring the overlay parameters

The dependency is visualized by a green frame around the referencing view. The green frame is referred to as selector. The user can link instances of both view types by applying this selector.

Instances of a parameter view and image view will be associated, if the following conditions are met:

  • The related instances of the image view and the parameter view refer to the same group (e.g. inspection) as indicated in the view title

  • The image view featuring the target image is framed in a green color

Custom Overlays

The custom overlay parameters are managed in the parameter view whereas the overlay shape is superimposed on an image featured in the image view.

This view type includes the following specific components:

  • Overlay list (center left)

  • Overlay parameters (right)

  • Parameter view controls (bottom left)

Overlay Parameters

This section of the parameter view displays the parameters for a selected overlay.

→ To display the overlay parameters, click on button Editnext to the overlay's label in the overview list.

Following overlay parameters are available:

Parameter/ Field Description
X X coordinate value [px] of the shape
X Variable X coordinate variable of the left corner of the shape
OFFSET LEFT Left position of X [px] relative to the left edge of the featured image
Y Y coordinate value [px] of the shape
Y Variable Y coordinate variable of the shape
OFFSET TOP Top position of Y [px] relative to the left edge of the featured image
WIDTH Width of the shape [px]
WIDTH VARIABLE Variable to determine the width of the shape
HEIGHT Height of the shape [px]
HEIGHT VARIABLE Variable to determine the height of the shape
COLOR Hexadecimal color of the shape, e.g. #ebfc02
COLOR VARIABLE Variable to determine the color of the shape
LINE WIDTH Line width of the shape [px]
LINE STYLE

Line style of the rectangle [px]. The following line styles can be applied:

  • line
  • dotted

Add Custom Overlay

You can create as many overlays for a target image as desired by adding them to the overlay list in the respective parameter view.

  • In the view context menu, click on the button Add Overlay

    — A new overlay will be added to the overlay list

Delete Custom Overlay

Besides the option of hiding an overlay in the image view, you can also delete an overlay

  1. In the overlay list, click on the button Edit overlay next to the overlay (label) to be deleted

    — The parameters are displayed right next to the overlay list

  2. In the view context menu, click on the button Delete Overlay

    — The overlay and its parameters are deleted
