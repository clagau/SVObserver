Liste

DEUTSCH

This view type features parameters in list format.

The user can view parameters available for the group (i.e. inspection) selected for the active view.

The List view includes the following specific components:

  • List of parameters in either full list display or select list display (center)

  • List view controls (bottom left)

Full List Display

The full list shows all available parameters for the selected group.

The list item indicates the fully qualified name for one parameter.

Parameter values can only be viewed in select list display.

Therefore the user has to select either one, several, or all list items by checking the checkboxes next to the listed parameters. The parameters thus selected can then be viewed in the selected list display.

Select List Display

The select display list displays a fully qualified name for the parameter including its (result-)values in table format.

The list will only show those parameters the user has previously selected in full list display.

List View Control

The list view currently one view type specific controls:

Element Description

Toggle between full list and select list display
Select parameters for select list display

Before you start: Make sure that the parameters are displayed in full list display.

To learn about how to add content, read instruction: Toggle between list displays

You can either select individual list items from the list or select all items at once.

Select individual parameters

  • In full→ list display, click on the checkboxes next to the list items

    — The checkbox is checked and the list item highlighted in a gray color

Select all parameters

  • In full list display, click on the checkbox in the list header

    — All checkboxes are checked and all list items highlighted in a gray color

Toggle between list displays

→ To switch from full list to select list display, you will have to select those parameters in the full list display you would like to be displayed in select list display.

To learn about how to add content, read instruction: Select parameters for select list display

  • In the view context menu, click on button Toggle

    — The list display will switch to the other list display
