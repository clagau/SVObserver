Image Strip

The image strip features the images, e.g. reject images. The newest image will be featured as the first image of the on the right.

If a view of the type image strip view has just been added to the view set, the user will have to choose an image source. The selected image will then be featured in the active view.

Image Strip View Controls

The image strip view offers a set of view type specific controls (from left to right):

Element Description

Pause
Pause image strip in image strip view

  • In the view context menu, press button Pause

    — The images will move in the direction from right to left

  • In the view context menu, press the button Play

    — The images will move in the direction from right to left
