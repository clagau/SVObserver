The image view features images acquired by the SVObserver and will be displayed with overlays by default.

This view type includes the following specific components:

  • Featured image (center), depicted with or without overlays

  • Image view controls (bottom left)

Click on image to enlarge

Note: You can save all view sets and incorporated views via the view manager. All changes will only be saved in the web app configuration.

Featured Image

If a view of the type image view has been added to the view set, the user will have to choose a display image. The selected image will then be featured in the active view.

→ To select a display image, click on the desired tool image listed in the center of the view

Click on image to enlarge

Dragging and Panning Images

The user can navigate the image within the view using the mouse.

Multi-touch is not supported as of yet.

Zoom image in image view

  1. Place the cursor on the image

  2. Use the mouse wheel to zoom the image

Drag image in image view

  1. Place the cursor on the image

  2. Hold down the mouse button and drag the image

Overlays in Image View

In the image view, the overlay is superimposed on the featured image. The overlays' parameters are managed in a separate view of the view type parameter view.

Overlay List in Image View

The overlay list offers options to show and hide overlays on the image.

Overlays can be shown or hidden either in total or individually (per overlay).

→ To open the overlay list, click on button Overlay list in the view controls bar.

The overlay list will open as in a separate window within the active view next to the featured image.

Overlay List Elements

The overlay list offers the following controls (from left to right):

Element Description

[Label]

 Overlay name
Hide individual overlay
Show individual overlay

Show all overlays on featured image

 

Hide overlays in image view

There are two options for hiding overlays in the featured image:

Hide all overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will open within the active view

  2. Click on button Hide all overlays

    — The overlays will no longer be displayed with the featured image

Hide individual overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Uncheck the checkbox next to the overlay to be hidden

    —The respective overlay will no longer be displayed with the image

Show overlays in image view

There are two options to display overlays in the featured image:

Show all overlays

  1. In the view context menu, click on button overlay list

    — The overlay list will be indicated in the active view

  2. Click on button the button Show all overlays

    — The overlays will be displayed with the image

Show individual overlays

  1. In the view context menu, click on the button Overlay list

    — The overlay list will be indicated in the active view

  2. Uncheck the checkbox next to the overlay to be hidden

    —The respective overlay will be displayed with the image

Image Information in Image View

The user can view information on the image and, if applicable, on the cursor position.

→ To view image information, click on button Image information in the view controls bar

The information will be displayed in a separate window within the active image view.

The user can view the size of the image (width/ height), cursor position (X/ Y) relative to image, and color and RGB color value (0, 0, 0) of the pixel.
