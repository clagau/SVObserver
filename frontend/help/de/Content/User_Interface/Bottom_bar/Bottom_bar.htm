DEUTSCH

The bottom bar includes the following components:

  • View set navigation
  • Menu

View set navigation

The left side of the bottom bar indicates one or more view set buttons. Each button relates to an individual view set and is labeled with the respective view set name. A gray-shaded button indicates that the referring view set is being displayed at the center of the page.

For more information on the view sets, read topic View Set

Menu

The menu is available on the right side of the bottom bar.

→ To open the menu, click on the button Menu

For more information on the menu, read topic Menu
