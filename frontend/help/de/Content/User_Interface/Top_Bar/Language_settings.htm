This area displays the currently applied interface language. The language settings can be changed by the user.

The display language is set to English by default.

The following languages are available:

en: English

de: German

Change Language

  1. Click on the button labeled with the language code

    — A list of available languages is be displayed

  2. Choose the designated language

    — The display language will be changed
