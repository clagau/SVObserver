The SVObserver can be operated in various modes. The choice of mode depends on the required tasks and processes conducted by the SVObserver.

This button in the top bar indicates the mode in which the SVObserver is operating. The authorized user can place an operating mode.

Note: The area in the top bar displays the operating mode which is currently set

 

The following modes are available:

Mode Description
The SVObserver will acquire and process images, read inputs and set outputs in this mode
The SVObserver will not accept any triggers while in this mode. No editing of tools or configuration can be done in this mode
The SVObserver will process previously archived images
The authorized user can access some tool adjustments while taking images for testing purposes
The authorized user can add, adjust and delete configurations and tools

 

Place Operating Mode

To conduct certain user interactions, you will have to place the SVObserver on the required operating mode.

When you change the mode, the SVObserver will be executed in the selected mode.

  1. Click on the button indicating the respective Operating Mode currently set

  2. Select the designated mode from the available sub-menu items

    — The operating mode will be set accordingly
