View Set Manager

You find the view set manager on the right side of the bottom bar.

Hover over image to enlarge

→ Click on the button View Set Manager to access the menu items.

→ A pop-up window appears in which you can manage your view sets.

 

You can create, rename, save, delete, export and import view sets in the view set manager.

 

View Set Manager Controls

The view set manager includes the following specific components and controls:

Element Description

Apply changes
Add new view set
Delete view set

Save all view sets and settings (organization of views, zoom factor and position of image, overlays and much more)
Export current view set
Import view set

Note

Settings saved via the SVOWebApp interface are only saved in the SVOWebApp configuration and have no effect on SVObserver. If you want to save your settings in the SVObserver configuration, you can do so by using the SVObserver interface.

Add new view set

→ Click on the button View Set Manager (bottom right corner of the screen).

→ Enter in the input field Name the desired name for the view set you want to create. The maximum length for the name is 20 characters.

→ Click on the button Add View Set.

→ The new view set appears in the bottom bar.

Rename existing view set

→ In the view set navigation (bottom bar), click on the button indicating the view set name you want to rename

→ The selected view set is displayed in the center of the screen. The view set button is highlighted in gray.

→ Click on the button View Set Manager (bottom right corner of the screen).

→ The view set manager opens in a pop-up window.

→ In the input field Name, replace the old title with the desired new title.

→ Click on the button Apply changes .

→ The selected view set is renamed. You can see the new name in the bottom bar.

Save all view sets

→ Click on the button View Set Manager (bottom right corner of the screen).

→ The view set manager opens in a pop-up window.

In the input field "Name", the selected view set is indicated.

→ Click on the button Save All View Sets .

→ All changes to the view sets are saved in the web app configuration.

→ When you restart the web app, the saved configuration is loaded automatically.

 Note

Settings saved via the SVOWebApp interface are only saved in the SVOWebApp configuration and have no effect on SVObserver. If you want to save your settings in the SVObserver configuration, you can do so by using the SVObserver interface.

If you edit a view set and want to close the SVOWebApp, the software reminds you that your changes have not yet been saved. The name of the configuration is written in italics and followed by an asterisk, if there are unsaved changes.

Export view set

→ Click on the view set button in the bottom bar indicating the name of the view set you want to export.

→ The selected view set is displayed in the center of the screen.

→ The view set button is highlighted in gray.

→ Click on the button View Set Manager (bottom right corner of the screen).

→ The view set manager opens in a pop-up window.

In the input field Name, the selected view set is indicated.

→ Click on the button Export current view layout .

→ If the export is successful, the view set file is saved in the Downloads folder of your browser.

Import view set

→ When you import a view set, the currently selected view set is overwritten. If you do not want to overwrite a view set, you must create a new one by adding a view set.

→ Click on the view set button in the bottom bar indicating the name of the view set you want to overwrite.

→ The selected view set is displayed in the center of the screen.

→ The view set button is highlighted in gray.

→ Click on the button View Set Manager (bottom right corner of the screen).

→ The view set manager opens in a pop-up window.

In the input field Name, the selected view set is indicated.

→ Click on the button Import a view layout from file .

→ Select a file in the Explorer window and click on Open.

→ A warning appears if you try to overwrite an existing view set. Click on OK if you want to overwrite the content of the view set. Click on Cancel if you do not want to overwrite the content of the view set.

→ After a successful import, a pop-up window appears confirming the import.

Delete view set

→ Click on the view set button in the bottom bar indicating the name of the view set you want to delete

→ The selected view set is displayed in the center of the screen.

→ The view set button is highlighted in gray.

→ Click on the button View Set Manager (bottom right corner of the screen).

→ The view set manager opens in a pop-up window.

In the input field Name, the selected view set is indicated.

→ Click on the button Delete View Set .

→ The selected view set is removed from your screen.
